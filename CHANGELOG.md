# Change Log

All notable changes to the "Tab Group Saver" extension will be documented in this file.

## [1.0.0] - 2023-10-01

### Added

-   Initial release of Tab Group Saver
-   Save currently open tabs as a named group
-   Restore any saved tab group with one click
-   View, edit, and delete saved tab groups
-   Set a default tab group to auto-restore on startup
-   Auto-save on close option
-   Multiple storage options (global, workspace, file)
-   TreeView UI in the activity bar
-   Keyboard shortcuts for common operations
